Mandelbrot

As my last project for Wats 3020, I designed a web page that would generate an image of a Mandelbrot set. Boundaries are created by the user through sliders and I used JavaScript to find the fractal components, draw the fractal to the screen, and create a loading spinner.

FizzBuzz

In Wats 3020, we were given the standard Fizz Bizz logic puzzle to solve with loops. The criteria we were given was:

  • Request a number from your user
  • Re-prompt user if the number is not a suitable number
  • Iterate from 1 to the user’s number (it becomes max)
  • For each number:
  • If the number is divisible by 3, print Fizz
  • If the number is divisible by 5, print Buzz
  • If the number is divisible by both 3 and 5, print FizzBuzz
